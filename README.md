# Material 3 Expressive Demo

A lightweight, responsive web demo showcasing Material 3 expressive design principles. Built with Bootstrap 5 and Vanilla JavaScript, it features interactive UI elements, smooth CSS animations, and mock JSON data integration.

## ✨ Features
- **Expressive UI**: Soft gradients, springy transitions, and layered shadows.
- **Interactive Cards**: Click buttons to dynamically display randomized affirmations or rephrased text.
- **Responsive Design**: Fully fluid layout that adapts to all screen sizes.
- **Optimized Performance**: Zero heavy dependencies (no jQuery); pure Vanilla JS and Bootstrap 5.
- **Custom Footer**: Branded footer with Bootstrap Icons and a hover-animated badge.

## 🛠️ Technologies Used
- **HTML5**
- **CSS3** (Custom animations & Bootstrap 5 utilities)
- **Vanilla JavaScript** (ES6+)
- **Bootstrap 5** (Grid, Flexbox, Components)
- **Bootstrap Icons**

## 🚀 Getting Started

1. **Clone the repository**:
```bash
   git clone https://github.com/khade217/material-3-expressive-demo.git
   cd material-3-expressive-demo
```

2. **Run the project**:
   Simply open the `index.html` file in your preferred web browser. No build step or local server is required.
